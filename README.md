# Door Handle Detection Project

## Description
This project implements an object detection model using PyTorch to detect 
door handles in images.  
It uses Faster R-CNN and custom dataset preprocessing.


## Project Files
- `model.py` → defines the model architecture
- `trainer.py` → training and validation functions
- `dataset.py` → dataset loader
- `main.py` → entry point to run training
- `args.py` → configuration and arguments
- `.gitignore` → ignores unnecessary files
